About Course

Front-end web development is everything involved in programming the user interface of a web application. Typically it refers to the Hypertext Markup Language(HTML), Cascading Style Sheets(CSS) and JavaScript portion of website production as opposed to database or server-sdie programming. It encompasses everything from building a simle page of HTML text to creating complex, responsive HTML5 websites designed to be accessed via various browsers, devices and screen sizes.
